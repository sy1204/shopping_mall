# Notion 문서 보관함

이 폴더는 프로젝트 진행 중 생성된 문서들 중 **Notion에 업로드하여 영구 보존하거나 팀원과 공유할 문서들**을 정리하는 곳입니다.

## 사용 가이드
1.  **목적**: Notion 페이지 구조에 맞춰 정리된 마크다운 파일들을 보관합니다.
2.  **형식**: Notion에 '복사/붙여넣기' 했을 때 깨짐 없이 잘 호환되도록 마크다운 표준을 준수합니다.
3.  **관리**: 에이전트가 작성한 리포트나 기획서 중 중요한 산출물은 이곳으로 복사하거나 이동하여 관리합니다.

## 주요 문서 목록 (예시)
- `29cm_analysis_report.md`: 경쟁사 벤치마킹 보고서
- `project_charter.md`: 프로젝트 헌장 및 그라운드 룰
- `meeting_notes.md`: 회의록
