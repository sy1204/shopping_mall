Logic of Legacy

Est. 2024

패션은 순간이지만 스타일은 영원하다는 말처럼, 우리가 추구하는 것은 일시적인 유행이 아닙니다. ‘Logic of Legacy’는 과거의 장인정신을 현대적인 미니멀리즘으로 재해석하는 과정입니다.

불필요한 장식을 덜어내고, 소재 본연의 질감과 실루엣에 집중합니다. 거친 린넨과 부드러운 실크의 조화, 구조적인 재단과 유려한 드레이핑의 공존. 이것이 [N-D]가 제안하는 클래식의 논리입니다.
