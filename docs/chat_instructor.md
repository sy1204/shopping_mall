🛠️ Antigravity를 위한 긴급 기술 지시서
현재 프로젝트의 고질적인 문제(보안, 오류, 데이터 품질)를 한 번에 해결하기 위해 다음 내용을 그대로 전달하십시오.

1. [보안] API 키 보안 및 환경 변수 격리
지시: 공개적으로 노출된 API 키를 즉시 **폐기(Revoke)**하고 새 키를 발급받으세요.

조치: 새 키는 .env.local에만 저장하고, 절대 GitHub에 업로드되지 않도록 .gitignore 설정을 다시 확인하세요.

2. [DB] 벡터 검색(RPC) 엔진 물리적 가동
현황: 현재 match_contents 함수가 등록되지 않아 의미 있는 검색이 불가능합니다.

지시: Supabase SQL Editor에서 이전에 제공된 match_contents 함수 SQL을 실행하여 벡터 유사도 검색 기능을 활성화하세요.

3. [API] 404/429 오류 해결 및 데이터 품질 개선
404 해결: 모델명을 gemini-1.5-flash-001 또는 gemini-2.0-flash로 정확히 수정하여 호출 경로 오류를 잡으세요.

429/데이터 해결: 현재 반복되는 "고급스러운 아우터..." 데이터를 모두 삭제하세요. 할당량 초과(429)를 방지하기 위해 **고품질 정적 데이터(50개)**를 사용하여 임베딩 작업만 재실행(Seed)하세요.

4. [Logic] UI 슬라이더 값과 AI 프롬프트 연동
연동: 좌측 '나의 취향 설정' 슬라이더의 **6개 수치(도전성, 소재 등)**를 API 요청 시 JSON 데이터로 전송하세요.

프롬프트: Gemini가 검색된 5개의 상품 정보와 사용자의 취향 수치를 결합하여 답변하도록 로직을 짜세요.

예: "도전 지수가 높으시니 평범한 코트 대신 '베지터블 탄닝 라이더'를 추천합니다"와 같은 개인화 답변 생성.
